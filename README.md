## Resume data fields

CV içeriği `src/data/resume-data.tsx` üzerinden yönetilir.

- `summary`: CV sayfasındaki (detaylı) özet
- `homeSummary`: ana sayfadaki (kısa/genel) özet
- `cities`: ana sayfada `Konya / İstanbul / Bursa` gibi gösterilen şehir listesi
- `workModes`: çalışma şekli tercihleri (`onSite`, `hybrid`, `remote`) — ana sayfada ikonlu rozetler olarak gösterilir
